Julia Set Explorer (GPU Edition)

This is a Julia set explorer running as a shader on your GPU. The full controls require a keyboard and mouse.

It’s a naive implementation that doesn’t allow for arbitrary precision. This means that it loses resolution at a relatively shallow zoom depth of ~100000x.

If you take any cool screenshots, feel free to share them with me. I love seeing how people use tools like this.

Controls

  • Click: set the image center
  • Scroll: zoom in / out
  • C: Increment color palette (decrement with Shift + C)
  • D: Increase pixel density (decrease with Shift + D)
  • E: Increase set exponent (decrease with Shift + E)
  • I: Increase constant imaginary component (decrease with Shift + I)
  • L: Show / hide command labels
  • O: Reset center to origin
  • R: Increase constant real component (decrease with Shift + R)
  • Z: Zoom in slowly (zoom out with Shift + Z)
  • ←→↑↓: Fine-adjust image center
  • Space: Play / pause animation (change direction with Shift + Space)
  • ?: Show this info pane
  • Esc: Hide this info pane

Note: not everything is configurable from this interface. To make deeper tweaks, follow the README and start running the code locally.
